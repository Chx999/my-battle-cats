# my-battle-cats

A minimal Java + LibGDX starter project inspired by The Battle Cats.

## Current structure

- `core`: game logic and battle screen
- `lwjgl3`: desktop launcher
- `assets`: game assets

## Implemented (starter)

- Basic battle screen
- Press `A` to spawn ally unit
- Press `L` to spawn enemy unit
- Units move left/right on a single lane
